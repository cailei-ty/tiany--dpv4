## ⚡ What is Tianyan?

Tianyan is **not another optimizer**. It is a **meta-optimization framework** that:

- **Unifies** all algorithms (SHADE, CMA-ES, DE, etc.) into a **16-dimensional continuous gene space**.
- **Synthesizes** new optimization strategies on the fly, rather than selecting from a fixed menu.
- **Separates signal from noise** using the **Taiji-field (太极场)** credit assignment system.
- **Adapts its exploration-exploitation balance** through a **four-layer Potential Field (势能场)** inspired by Yin-Yang dynamics.
- **Routes tools and strategies** via a **Wu-Xing (五行) engine** that maps philosophical elements to functional modules.

The result: **cross-domain robustness, full explainability, and zero pretraining required.**

| Layer | Name | Core Mechanism |
|-------|------|----------------|
| **L7** | Philosophical Ontology | Yin-Yang, Taiji, Bagua, Wu-Xing → technical mappings |
| **L6** | Information Bus (TaoFlow) | 11-dimensional global state vector, observer pattern |
| **L5** | Agent Cognitive Layer | Task planning, 5-level memory, LLM flywheel, safety expert |
| **L4** | Meta-Optimization (MetaBrain) | Dual-brain architecture, GeneONN, Taiji Field, gene synthesis |
| **L3** | Optimization Execution | Potential Field (4-layer PID), MAB scheduler, feature detection |
| **L2** | Plugin Ecosystem | 18+ operators, 8 standard optimizers, 6 domain plugins |
| **L1** | Service Adaptation | FastAPI, math advisor, LLM inference |
| **L0** | Adversarial Evolution | Scenario generator, recipe evolver, feature evolver |

## 🔬 Why Tianyan?

| | Traditional AutoML (Optuna, SMAC) | Tianyan |
|------|-----------------------------------|---------|
| **Approach** | Discrete algorithm selection | Continuous algorithm synthesis |
| **Credit assignment** | Simple win-rate | Taiji-field signal/noise separation |
| **Cross-domain ability** | Requires retuning | Demonstrated across chip design, finance, ballistics, tracking |
| **Explainability** | Low | Every decision traceable to gene vector + credit score |
| **Resource requirement** | Moderate | Works on a single machine, cold-start |
